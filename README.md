# protonotif
Get protonmail notification on desktop

## Install

```
git clone https://github.com/jppaled/protonotif.git
cd protonotif
npm i install
npm start
```

## Dependencies
- node@10

## Npm package dependencies
- express
- fs
- node-notifier
- openpgp@4.10.10 (not @5 because of node@10
- path
- protonmail-api
